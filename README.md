# PostsListApp

An application that list posts from "JSONPlaceholder API".

Using Stack:

- Kotlin
- Android DataBinding
- AndroidX
- Mockk
- Retrofit2.0
- Modularization
- RxJava
- MVVM
- LiveData
- Glide
- Android ViewModel
- Android JetPack
- Android Arch Lifecycle
- Clean architecture
- Android Architecture Components
- Room
