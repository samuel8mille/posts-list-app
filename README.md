# PostsListApp

An application that list posts from "JSONPlaceholder API".

Using Stack:

- Kotlin
- Android DataBinding
- AndroidX
- Mockk
- Retrofit2.0
- RxJava
- MVVM
- LiveData
- Glide
- Android ViewModel
- Android JetPack
- Android Arch Lifecycle
